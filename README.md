# Мессенджер

## Быстрое начало

- Установка зависимостей
    - `npm install`
- Сборка и запуск сборки в локальном сервере
    - `npm run build`
    - `npm run preview`
- Или запуск локального dev сервера с hot-reload
    - `npm run start`
